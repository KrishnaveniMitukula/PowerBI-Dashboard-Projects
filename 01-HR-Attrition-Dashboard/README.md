## 📌 Project Overview
This project analyzes employee attrition data from 
IBM HR Analytics dataset containing 1,470 employees. 
The dashboard helps HR teams and management understand 
the key factors driving employee attrition and take 
data driven decisions to improve employee retention.

---

## ❓ Problem Statement
Organizations face huge costs when employees leave. 
This dashboard answers the following questions:
- Which department has the highest attrition?
- Which age group is most likely to leave?
- Does working overtime increase attrition?
- Does marital status affect attrition?
- Which job level has the most attrition?
- Do single employees leave more than married ones?
